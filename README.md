## Running Locally

- Clone this Repository

```bash
git clone https://github.com/Naveen-759/agri-credit.git
```

- Create `.env` file in root directory

```bash
MONGO_URL=
JWT_SECRET=
```

- Create `.env` file in client directory

```bash
VITE_FIREBASE_API_KEY=
```

- In `/` Run

```bash
 npm run dev
```

- In `/client` Run

```bash
 npm run dev
```
